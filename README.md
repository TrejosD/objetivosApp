# NewLeaf

Esta app mobile fue pensada para ayudarte a crear habitos nuevos.

- Colocas el nombre de tu nuevo habito.
- Luego cuantas veces quieres hacerlo mensualmente.
- El app te ayuda a llevar registro de tu avance, y cada mes se reinicia.
- Cada mes sera un nuevo capitulo de habitos por cumplir.

---

## Reglas Motivacion

1.Nunca castigar visualmente.
2.Celebrar micro-logros.
3.Crear sensacion de progreso visible.

## Los cambios en DB, se debe usar el siguiente comando

```
flutter pub run build_runner build
```

## Para cambiar el splash_screen

```
dart run flutter_native_splash:create
```

## Para cambiar el app package name

```
dart run change_app_package_name:main "com.tunombre.appname"
```
